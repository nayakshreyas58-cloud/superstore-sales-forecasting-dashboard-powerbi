# 📊 Superstore Sales Analysis & Forecasting Dashboard

## 🚀 Project Overview

This project showcases an advanced Power BI dashboard built using Superstore sales data to analyze business performance and forecast future sales trends. The dashboard provides interactive visualizations, key performance indicators (KPIs), regional insights, category-wise analysis, and a 15-day sales forecast using time-series forecasting techniques.

The primary objective of this project is to transform raw sales data into meaningful business insights that help stakeholders make informed decisions and identify growth opportunities.

---

## 🎯 Key Features

- Interactive Power BI Dashboard
- Sales, Profit, and Quantity Analysis
- Regional and State-wise Performance Tracking
- Category and Sub-Category Analysis
- Customer Segment Insights
- Payment Mode Analysis
- Shipping Mode Performance Evaluation
- Monthly Sales and Profit Trends
- 15-Day Sales Forecasting
- Dynamic Filters and Slicers for Data Exploration

---

## 🛠️ Tools & Technologies Used

- Power BI
- Microsoft Excel
- Data Cleaning & Transformation
- DAX (Data Analysis Expressions)
- Data Visualization
- Time-Series Forecasting
- Business Intelligence (BI)

## 📈 Dashboard Insights

The dashboard enables users to:

- Monitor overall sales performance
- Identify top-performing regions and categories
- Analyze customer purchasing behavior
- Track monthly profit and sales trends
- Compare performance across different segments
- Forecast future sales for the next 15 days
- Support data-driven business decisions
